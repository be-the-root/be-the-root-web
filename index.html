About Me

Hi, I'm Udesh. I am a 12-year-old security researcher and ethical hacker with a deep passion for web application security testing. I spend my time exploring web architectures, analyzing traffic, and hunting for software vulnerabilities in controlled laboratory environments like Hack The Box. Driven by curiosity about how things work under the hood, I focus on identifying security flaws—such as injection bugs and broken access controls—to understand how to better defend the digital world. I am constantly building my technical toolkit, exploring automation , and working toward finding real-world bugs to help make the web a safer place.
